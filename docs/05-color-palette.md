# 🌿 Clean Office Color Palette

A calm, minimal, professional workspace aesthetic with soft white, mint, and natural browns.

| Color Role            | Hex       | Description                             |
| --------------------- | --------- | --------------------------------------- |
| **Soft White (Base)** | `#F7F9F7` | Warm soft white for backgrounds         |
| **Mint Light**        | `#CFEDE3` | Airy mint for subtle accents            |
| **Mint Core**         | `#9FD8C6` | Main mint tone for UI elements          |
| **Mint Deep**         | `#5FAF9A` | Stronger mint for buttons or highlights |
| **Light Brown**       | `#C7A78B` | Soft natural wood tone                  |
| **Coffee Brown**      | `#8B6A52` | Grounding mid brown                     |
| **Deep Brown**        | `#4E3A2E` | Rich dark brown for text or contrast    |
| **Cool Gray Accent**  | `#AEB8B2` | Neutral balance color                   |

## Suggested Usage

- **Background:** `#F7F9F7`
- **Primary accent:** `#9FD8C6`
- **Secondary accent:** `#CFEDE3`
- **UI / cards:** `#FFFFFF` or `#F7F9F7`
- **Text:** `#4E3A2E`
- **Warm accents:** `#C7A78B`

## Quick Visual Balance Rule

- **60% Soft White**
- **25% Mint tones**
- **10% Brown tones**
- **5% Neutral gray**
